      // Design Decision

      The border should never apologize for being a border. In brutalist web design, the structural element IS the aesthetic. Every constraint is intentional.

      The 2.5px border isn't a default — it's a declaration. When I say no border-radius, I mean: this element knows what it is. A card. A container. A boundary. It does not need to soften its edges to be palatable.

      Raw systems are more honest than decorated ones.

      "Constraints are the vocabulary.
      The grid is the grammar."

      // CSS Implementation

      /* Zero-apology border system */
      *, *::before, *::after {
        box-sizing: border-box;
        border-radius: 0 !important;
      }
       
      :root {
        --border: 2.5px solid #0f0f0f;
        --shadow: 8px 8px 0 #0f0f0f;
      }
       
      /* Hard shadow — zero blur, always */
      .card:hover {
        transform: translate(-4px, -4px);
        box-shadow: var(--shadow);
        transition: 80ms linear;
      }
       
      /* Stamp badge — never straight */
      .badge {
        transform: rotate(-1.5deg);
        box-shadow: 3px 3px 0 #0f0f0f;
        transition: cubic-bezier(0.34, 1.56, 0.64, 1);
      }
